DataVision – Business Analytics Dashboard-
A simple, clean, and interactive dashboard that helps visualize business startup data using FastAPI (backend) and React + Recharts (frontend).

Project Summary-
DataVision is a full-stack project where:
The backend reads a CSV file, converts it into JSON & XML, cleans the data, calculates useful business metrics, and provides everything through an API.
The frontend fetches this data and displays it in a beautiful dashboard using React and Recharts.
